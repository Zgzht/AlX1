## 图片规格要求

- 图标尺寸：24x24px, 32x32px, 48x48px
- 头像尺寸：80x80px, 120x120px
- 分享图片：750x1334px
- 格式：PNG（支持透明背景）
- 颜色：主色调 #4CAF50（绿色）

## 使用说明

1. 所有图片文件应放在 `/miniprogram/images/` 目录下
2. 在代码中使用相对路径引用：`/images/filename.png`
3. 建议使用 2x 和 3x 分辨率的图片以支持高分辨率屏幕
4. 图片文件名使用小写字母和连字符，避免使用空格和特殊字符
